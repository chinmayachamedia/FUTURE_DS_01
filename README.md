# FUTURE_DS_01
Business Sales Performance Analytics
## Data Cleaning
Duplicate records were removed and date columns were standardized.

## Analysis Performed
- Sales trend analysis using PivotTables
- Category-wise sales analysis
- Region-wise sales and profit analysis

## Key Insights
- Sales show an increasing trend
- Technology category leads in sales
- West region performs best

## Recommendations
- Focus on high-performing categories
- Improve underperforming regions
- Optimize inventory planning
